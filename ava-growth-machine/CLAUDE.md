AVA AI est une plateforme d'agents vocaux IA qui répondent aux appels entrants et sortants pour les PME.

- `/source [ville] [vertical]` → Lance le scraping Google Maps pour cette ville/vertical

- `/personalize [lead]` → Génère un email personnalisé pour ce lead
- `/sequence [lead]` → Génère la séquence complète 3 emails pour ce lead
- `/audit [nom_etablissement]` → Lance un missed call audit sur cet établissement
- `/demo-script [vertical]` → Génère le script de démo pour ce vertical
- `/objection [objection]` → Retourne la réponse battle card
- `/weekly` → Génère le rapport hebdomadaire KPIs
- `/content [type] [sujet]` → Génère du contenu (linkedin/blog/video)
