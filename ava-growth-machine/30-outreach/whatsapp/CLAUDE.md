# WhatsApp — CLAUDE.md

## WhatsApp est critique pour les PME françaises. Beaucoup de décideurs PME préfèrent WhatsApp à l'email.

## Règles
- Max 3 messages. Pas de spam.
- Toujours se présenter dans le premier message.
- Pas de lien dans le premier message (risque blocage).
- Vocaux OK (max 30 secondes).
- Répondre dans les 2h si le prospect répond.
- Emojis : 1-2 max, jamais plus.

## Quand utiliser WhatsApp
- Lead score ≥ 80 qui n'a pas répondu à l'email
- Numéro mobile identifié (pas fixe)
- Post-démo : follow-up rapide
- Prospects dans des groupes WhatsApp professionnels

## Séquence WhatsApp (3 messages)

### Message 1 (J+5 après email sans réponse)
```
Bonjour {Prénom}, c'est Nissiel.

Je vous ai envoyé un email sur les appels manqués 
pour {nom_etablissement} — vous l'avez peut-être raté.

En 30 secondes : on a un assistant qui répond à vos appels 
24/7 et prend les RDV à votre place.

Je peux vous montrer rapidement ?
```

### Message 2 (J+3 si pas de réponse)
```
{Prénom}, un chiffre rapide :

Les {type} manquent ~30% de leurs appels.
Pour {nom}, ça peut représenter {estimation}€/an.

Voulez-vous que je vous montre en 5 min ?
Pas d'engagement, juste une démo rapide.
```

### Message 3 (J+5 — dernier)
```
Dernier message {Prénom} 🙂

Si le sujet des appels manqués n'est pas prioritaire, 
je comprends.

Répondez ici si ça change — je suis toujours dispo.

Bonne journée.
```

## Groupes WhatsApp (tactique organique)
- Rejoindre des groupes de professionnels locaux (dentistes [ville], directeurs d'école, agents immo)
- Ne JAMAIS pitcher directement dans le groupe
- Apporter de la valeur : répondre à des questions, partager des insights
- Quand quelqu'un mentionne un problème de téléphone/accueil → DM privé
